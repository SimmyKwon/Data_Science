This directory contains the codes that I worked on sentiment analysis on national garden in Korea (태화강 국가정원)
There are two folders in the directory, one folder collecting text data from instagram, twitter and naver, and the other that performs preprocessing, visualisation, and sentiment analysis on the processed data.
The method of analysis is using Korean language dictionary made by Kunsan National University in Korea, containing the stopwords.

This directory doesn't contain the codes fine-tuning and performing analysis using BERT model, as the codes got lost 😥😥.
